# EEG Brain Age Prediction with Machine Learning and Optimization

This repository implements machine learning models for EEG-based brain age prediction, combining Random Forest and Multi-Layer Perceptron (MLP) architectures with advanced optimization techniques.

##  Project Overview
Predicting biological brain age from EEG signals using ensemble machine learning approaches, optimized through both Genetic Algorithms and Bayesian Optimization methods.

##  Dataset
- **Source**: EEG Brain Age Dataset (preprocessed EEG recordings with age labels)
- **Features**: Multi-channel EEG time-series data
- **Target**: Chronological age prediction

##  Features

### Models Implemented
- **Random Forest Regression**
- **Multi-Layer Perceptron (MLP) Neural Networks**

### Optimization Techniques
- **Genetic Algorithm (GA)** for hyperparameter tuning
- **Bayesian Optimization** for model parameter optimization
- **Comparative analysis** of optimization strategies

### Key Features
- End-to-end EEG preprocessing pipeline
- Feature extraction and selection methods
- Cross-validation and model evaluation
- Visualization of results and feature importance
- Reproducible experimentation framework
